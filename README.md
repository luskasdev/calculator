# 🧮 Luskas Calc `v1.0.0`

> Uma calculadora ultra-minimalista com estética Glassmorphism, projetada para oferecer uma experiência fluida e elegante diretamente no navegador.

---

## 🛠️ O Projeto

O **Luskas Calc** nasceu da ideia de unir utilidade matemática com um design "Premium Dark". Diferente de calculadoras convencionais, este projeto foca em **animações suaves**, **feedback tátil visual** e uma interface limpa que não polui a visão do usuário.

### ✨ Funcionalidades Principais
- **Operações Completas:** Soma, subtração, multiplicação, divisão e parênteses.
- **Suporte a Teclado:** Digite diretamente para calcular (suporta `Enter`, `Esc` e `Backspace`).
- **Histórico Inteligente:** Visualize a expressão anterior logo acima do resultado atual.
- **Design Responsivo:** Adaptada para telas de desktop e dispositivos móveis.
- **Glassmorphism:** Interface translúcida com desfoque de fundo (backdrop-filter).

---

## 🚀 Tecnologias Utilizadas

Para garantir leveza e performance máxima, o projeto foi construído com:

- **HTML5 Semântico:** Estrutura sólida e acessível.
- **CSS3 Moderno:** Uso de variáveis, Flexbox, Grid e animações `cubic-bezier`.
- **Vanilla JavaScript:** Lógica pura, sem dependências externas, garantindo carregamento instantâneo.

---

## 🎨 Interface

O design utiliza a paleta **Cyber Dark**:
- **Background:** `#050505` (Deep Black)
- **Accent:** `#00f2ff` (Neon Cyan)
- **Glass:** `rgba(255, 255, 255, 0.03)`
